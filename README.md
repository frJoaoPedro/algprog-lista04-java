# 📚 Exercícios Práticos em Java

Este repositório reúne exercícios em Java focados em conceitos básicos de programação, como estruturas de repetição, condicionais, entrada de dados com `Scanner` e lógica matemática.

---

## 📂 Arquivos

### 🔐 Validação e Segurança
- **ValidadorDeNotas.java** – Valida notas entre 0 e 10.
- **NomeDiferenteDeSenha.java** – Impede que usuário e senha sejam iguais.
- **Validador.java** – Valida nome, idade, salário, sexo e estado civil.

### 📊 Cálculos
- **maiorNumero.java** – Encontra o maior entre cinco números.
- **mediaNumeros.java** – Calcula soma e média.
- **numerosImpar.java** – Exibe números ímpares de 1 a 50.

### 🔢 Sequências
- **numeros.java** – Mostra números de 1 a 20.
- **numerosInteiros.java** – Gera sequência entre dois números.

### 🌍 Simulações
- **Populacao.java** – Calcula tempo de crescimento populacional.
- **PopulacaoV2ex5.java** – Versão com entrada do usuário e validações.
